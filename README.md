# вҡҷпёҸ Installation Guide

### 1пёҸвғЈ Clone the Repository

```bash
git clone https://github.com/kaushik-ryn/SnapMyEat-Development.git
```

### 2пёҸвғЈ Navigate to the Project

```bash
cd SnapMyEat-Development
```

### 3пёҸвғЈ Create a Virtual Environment

```bash

```

### 4пёҸвғЈ Activate the Environment

**Windows**

```bash

```

**Linux / macOS**

```bash

```

### 5пёҸвғЈ Install Dependencies

```bash

```

### 6пёҸвғЈ Apply Database Migrations

```bash

```

### 7пёҸвғЈ Run the Development Server

```bash

```

```text

```
